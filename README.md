# 엄마를 위한 로또 번호 추첨기 ✨

> 💝 데이터 기반 스마트 추천으로 특별한 행운을

React + Vite로 만든 아름답고 직관적인 로또 번호 생성 웹 애플리케이션입니다.

---

## 🎯 주요 기능

- **3가지 추첨 방식** - 스마트 추천, 균형 조합, 완전 랜덤
- **실시간 통계** - 합계, 평균, 홀짝 비율, 연속 번호
- **아름다운 UI/UX** - 글래스모피즘 디자인과 부드러운 애니메이션
- **완벽한 반응형** - 모바일, 태블릿, 데스크톱 모두 지원

---

## 🎲 추첨 방식 비교

### 🎯 스마트 추천 (Smart Recommendation)

**작동 방식:**

- 과거 1~1200회차의 **실제 당첨 번호 빈도 데이터**를 기반으로 합니다
- 자주 나온 번호에 더 높은 가중치를 부여합니다
- 최근 50회차 트렌드에 있는 번호는 **추가로 +30 가중치**를 받습니다
- 가중치에 따라 확률적으로 번호를 선택합니다

**예시:**

```
번호 43: 과거 180회 출현 → 가중치 180
번호 43: 최근 트렌드 포함 → 가중치 210 (180+30)
번호 28: 과거 151회 출현 → 가중치 151
```

**특징:** 데이터 기반이라 "근거가 있는" 느낌을 주며 심리적 안정감을 제공합니다.

---

### ⚖️ 균형 조합 (Balanced Combination)

**작동 방식:**

- 1~45를 **5개 구간**으로 나눕니다:
  - 구간 1: 1-9
  - 구간 2: 10-18
  - 구간 3: 19-27
  - 구간 4: 28-36
  - 구간 5: 37-45
- 최소 **4개 이상의 다른 구간**에서 번호를 선택합니다
- 번호가 특정 구간에 몰리지 않고 고르게 분산됩니다

**예시:**

```
❌ [1, 2, 3, 4, 5, 10] - 구간 1, 2에만 몰림
✅ [3, 15, 22, 31, 38, 42] - 5개 구간에 골고루 분산
```

**특징:** 실제 당첨 번호는 고르게 분포하는 경향이 있어 "밸런스 있는" 조합을 선호하는 분들에게 적합합니다.

---

### 🎲 완전 랜덤 (Complete Random)

**작동 방식:**

- `Math.random()`을 사용한 **순수 무작위** 선택
- 아무런 규칙이나 데이터 없이 1~45 중에서 랜덤하게 6개 선택
- 중복만 제거하고 정렬합니다

**예시:**

```
[7, 12, 15, 23, 39, 44] - 완전히 랜덤
[2, 3, 4, 5, 6, 7] - 이것도 가능 (확률은 낮지만)
```

**특징:** 가장 "공정한" 방식입니다. 로또는 본질적으로 무작위이므로 이론적으로는 가장 정확한 방법입니다.

## 📊 추첨 방식 비교표

| 항목               | 스마트 추천      | 균형 조합          | 완전 랜덤          |
| ------------------ | ---------------- | ------------------ | ------------------ |
| **근거**           | 과거 데이터 분석 | 구간별 분산 전략   | 없음 (순수 무작위) |
| **심리적 안정감**  | ⭐⭐⭐⭐⭐       | ⭐⭐⭐⭐           | ⭐⭐               |
| **실제 당첨 확률** | 동일             | 동일               | 동일               |
| **추천 대상**      | 데이터를 믿는 분 | 균형을 중시하는 분 | 운에 맡기는 분     |

> **💡 재미있는 사실:** 실제로는 **세 가지 모두 당첨 확률은 완전히 동일**합니다!
>
> 로또는 매번 독립적인 추첨이므로 과거 데이터는 미래에 영향을 주지 않습니다. 하지만 **"왜 이 번호를 선택했는지"에 대한 이야기**가 있으면 더 재미있고, 막연한 불안감이 줄어듭니다.

## 🚀 시작하기

### 필수 요구사항

- Node.js 18.0 이상
- npm 또는 yarn

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 미리보기
npm run preview
```

개발 서버는 기본적으로 `http://localhost:5173`에서 실행됩니다.

## 🎯 주요 통계

생성된 번호에 대해 다음 통계를 실시간으로 제공합니다:

- **합계:** 6개 번호의 총합
- **평균:** 합계 ÷ 6 (소수점 첫째자리)
- **홀짝 비율:** 홀수 개수 : 짝수 개수
- **연속 번호:** 차이가 1인 인접 번호 쌍의 개수
